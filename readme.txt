= // Social Stats Plugin - Daily Social Signals Tracker! =

- Homepage Signals Count - How many times your homepage has been shared or liked on the social networks.

- Signals per Post - See in total, how many social signals have your wordpress posts generated. The Plugin updates all your signals and goes through all your posts DAILY! This gives you the ability to Optimize your "Call to Share" and test different ways to generate social signals.


== Installation ==

Using the WordPress dashboard

1. Login to admin panel
2. Go to Plugins
3. Select Add New
4. Search Social Stats Panel
5. Select Install Now
6. Select Activate Plugin

Manual

1. Download the plugin and extract the files
2. Upload the directory "social-stats-panel" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Social Stats" Tab created by this plugin for your Social Signals Stats

That's all. You dont need to do anything else. Enjoy!

== Frequently Asked Questions ==

= How do you update the stats? =

We use Wordpress cron to automatically update your social stats EVERY DAY!

= Which social networks are supported? =

Currently only Facebook, Google Plus and Twitter are supported. (Major ones!)
